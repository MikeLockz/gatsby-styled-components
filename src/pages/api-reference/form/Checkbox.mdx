# Checkbox documentation

Use the `Checkbox` component when users can choose 0 or more than 1 options from a list. `Checkbox` maps to a checkbox input and a label.

*Want to understand best practices for using  `Checkbox`?* [Read the design guidelines](https://consensys.github.io/rimble-ui/?path=/story/components-form-checkboxes--design-guidelines)

**Import component**

    import { Checkbox } from 'rimble-ui';

## Examples

Different ways to use the `Checkbox` component

### Default checkbox

<Checkbox
label="Checkbox default"
required="true"
/>

    <Checkbox
      label="Checkbox default"
      required="true"
    />

### Checked checkbox

<Checkbox
checked
label="Checkbox checked"
required="true"
/>

    <Checkbox
      checked
      label="Checkbox checked"
      required="true"
    />

### Disabled checkbox

<Checkbox
disabled
label="Checkbox disabled"
required="true"
/>

    <Checkbox
      disabled
      label="Checkbox disabled"
      required="true"
    />

### Checked and disabled

<Checkbox
checked
disabled
label="Checkbox checked disabled"
my={2}
required="true"
/>

    <Checkbox
      checked
      disabled
      label="Checkbox checked disabled"
      my={2}
      required="true"
    />
