# Input documentation

The `Input` component provides a styled version of HTML `<input>`.

*Want to understand best practices for using `Input`?*  [Read the design guidelines](https://consensys.github.io/rimble-ui/?path=/story/components-form-inputs--design-guidelines)

**Import component**

    import { Input } from 'rimble-ui';

## Examples

Different ways to use the `Input` component in your product. Note: these examples use the `[Field](https://consensys.github.io/rimble-ui/?path=/story/components-form-field--documentation)` component to render associated labels.

### Placeholder

Adds placeholder text to suggest the type of input expected.

<Input type="text" required="true" placeholder="e.g. My hodl wallet" />

    <Input type="text" required="true" placeholder="e.g. My hodl wallet" />

### Disabled input

<Input type="text" placeholder="e.g. My hodl wallet" required="true" disabled />

    <Input type="text" placeholder="e.g. My hodl wallet" required="true" disabled />

### Optional input

Renders "(Optional)" next to the label.

<Input type="text" placeholder="e.g. My hodl wallet" />

    <Input type="text" placeholder="e.g. My hodl wallet" />

### Text input

Brings up a standard keyboard.

<Form.Input type="text" required="true" placeholder="e.g. 0xAc03BB73b6a9e108530AFf4Df5077c2B3D481e5A" />

    <Form.Input type="text" required="true" placeholder="e.g. 0xAc03BB73b6a9e108530AFf4Df5077c2B3D481e5A" />

### Email input

Give you an @ on your keyboard.

<Form.Input type="email" required="true" placeholder="you@example.com" />

    <Form.Input type="email" required="true" placeholder="you@example.com" />

### Password input

Masks any input.

<Form.Input type="password" required="true" />

    <Form.Input type="password" required="true" />

### Number input

Opens the number page of your keyboard.

<Form.Input type="number" required="true" placeholder="e.g. 123" />

    <Form.Input type="number" required="true" placeholder="e.g. 123" />

### Search input

Allows users to quickly remove a search term.

<Form.Input type="search" required="true" placeholder="e.g. Ethereum" />

    <Form.Input type="search" required="true" placeholder="e.g. Ethereum" />

### Telephone input

Brings up the telephone keyboard.

<Form.Input type="tel" required="true" placeholder="e.g. +445575242519" />

    <Form.Input type="tel" required="true" placeholder="e.g. +445575242519" />

### URL input

Bring up a url keyboard.

<Form.Input type="url" required="true" placeholder="e.g. rimble.consensys.design" />

    <Form.Input type="url" required="true" placeholder="e.g. rimble.consensys.design" />
